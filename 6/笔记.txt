form 最重要的 通信方式
1. action  提交到哪儿
2. method  GET/POST/PUT/DELETE/HEAD
	GET ：数据放大URL中传输 / 数据量很少，缓存，看得见   <= 32k
	POST: 放到 body 里面   / 数据量比较大，看不见，不会缓存  <= 1G


	GET ---- 获取数据
	POST ,PUT  ---- 发送数据
	DELETE   ------ 删除东西
	HEAD ----- 让服务器只发送头回来就行 (不需要内容)

3. name  --- 名字

4. enctype

	application/x-www-form-urlencoded  名字=值&名字=值    默认，小数据
	multipart/form-data                分块              文件上传，大数据

ajax 原理 --- XMLHttpRequest



readstate
0    初始状态   xhr对象刚创建完
1    连接       连接到服务器
2    发送请求    刚刚 send 完
3    接收完成    头接收完了
4    接收完成    body 接收完了


status ----  http 状态码
1xx   消息
200   成功
3xx   重定向
	301 Moved Permanently  永久重定向 -- 下次不会再请求这太服务器了
    302 Move temporarily   临时重定向 -- 下回依然会请求这台服务器
    304 Not Modified       缓存
    4xx                    请求错误
    5xx                    服务器错误
    6xx+                   自定义
